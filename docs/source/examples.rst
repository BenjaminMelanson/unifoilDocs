Examples
========

.. autosummary::
   :toctree: generated

UniFoil comes with numerous examples to help show how the data is stored, accessed, and utilized.
Unlike typical codebases, our test scripts **test<0-8>.py** (see the `Installation page <https://unifoildocs.readthedocs.io/en/latest/installation.html>`_for more details) provide all the example usage the end--user needs.
